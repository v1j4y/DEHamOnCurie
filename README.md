DEHam on Curie
==============


Scripts and useful files to help using
DEHam on Curie cluster.

Usage
-----

1. Modify initial input file `tJ_heis4.inp`
2. Create directories

```
./python create.py
```

3. Copy executable and bash script

```
./python submit.py
```

4. Submit each input file

```
sbatch job.sh
```
